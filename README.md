## Featured Projects

### FreelasBot — AI Automation for Freelance Marketplaces
> NestJS · React · Playwright · Ollama · WebSockets · SQLite

Full-stack application for automating job applications on marketplaces. Resilient scraping with Playwright, AI-generated proposals via local LLM (Ollama), execution pipeline with state control, job deduplication, and real-time communication via WebSockets.

---

### MonteSanto App — Dairy Logistics Management
> Flutter · Dart · Riverpod · GoRouter · SQLite

Cross-platform mobile app (iOS/Android) for digitizing the milk collection and distribution chain. RBAC access control, quality parameters in the field, end-to-end traceability, and robust offline support with persistent queue and automatic sync.

---

### Intelligent Analysis & Automation System
> FastAPI · LangChain · LLMs · Embeddings · RAG

Backend for decision support automation. Integrated LLMs, embeddings and RAG architecture — reduced manual review cycles by ~60% and increased operational efficiency.

---

### BoneVision 3D — Educational Osteology Software
> React · Supabase

Interactive educational software for animal osteology study, adopted by 3 university departments.
